# Alex Ward

Contains the source code for my personal page/blog which is published
via Github pages and available at https://alxwrd.co.uk/


## Getting started

```shell
git clone git@github.com:alxwrd/alxwrd.github.io.git && cd alxwrd.github.io

pnpm install
pnpm dev
```
